Data Analytics Capstone Project

This is the repository for my capstone project.
In this project I am aiming to observe differences between audience and critic review scores on Rotten Tomatoes.
By analyzing the scores, and the box office gross of movies, I want to determine which score is a better indicator of a higher grossing film.


The dataset was found through Kaggle, linked here: https://www.kaggle.com/datasets/thedevastator/rotten-tomatoes-top-movies-ratings-and-technical 

A number of transformations will be applied to clean the data and prepare it for an analysis tool.
